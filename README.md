# Ormula-Ten 🏎️🔥

Welcome to **Ormula-Ten**, a thrilling retro-inspired 2D racing game built with pure web technologies! Dive into fast-paced, classic racing action right in your browser — no installs, no fuss, just pure speed and fun.

---

## 🚀 Features

- 🎮 **Classic 2D Racing Gameplay** — Simple controls, addictive mechanics, and smooth animations.
- 🌐 **Browser-Based** — Play instantly on any modern browser without setup.
- ⚡ **Lightweight & Fast** — Built with HTML, CSS, and JavaScript for quick load times.
- 🎨 **Customizable Assets** — Easily tweak graphics and styles to make it your own.
- 🛠️ **Open Source & Extensible** — Perfect for developers wanting to learn or contribute.

---

## 📂 Project Structure

| File/Folder         | Description                                   |
|---------------------|-----------------------------------------------|
| `index.html`        | Main entry point — loads the game interface. |
| `game.js`           | Core game logic and mechanics.                 |
| `game_additions.js` | Extra features and game enhancements.          |
| `style.css`         | Styling for the game UI and layout.            |

## 🏁 Getting Started

### 1. Clone the repository
git clone https://github.com/j-41/Ormula-Ten.git
cd Ormula-Ten

### 2. Open the game
- Option A: Double-click index.html or drag it into your browser.
- Option B (Recommended): Use a local development server for best performance and to avoid browser restrictions.

### 3. Run a local server (optional but recommended)
Using Python 3:
python -m http.server 8000

Using Node.js with http-server:
npm install -g http-server
http-server

Then open your browser at http://localhost:8000 (or the address shown).

| Action          | Key(s)               |
|-----------------|----------------------|
