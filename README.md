# Ormula-Ten

Ormula-Ten is a classical 2D racing game built using web technologies. It is a lightweight, browser-based game designed to capture the nostalgia of retro 2D racing experiences.

---

## Features

- **Classic 2D Gameplay:** Simple, engaging racing mechanics.
- **Web-Based:** Runs directly in your modern web browser.
- **Lightweight:** Built with HTML, CSS, and JavaScript, ensuring fast load times.

---

## Project Structure

| File/Folder         | Description                                |
|---------------------|--------------------------------------------|
| `index.html`        | The main entry point for the game.         |
| `game.js`           | Core game logic and engine.                 |
| `game_additions.js` | Supplemental features and extensions.      |
| `style.css`         | Styling for the game interface and layout. |
| `src/`              | Contains game assets like logos and graphics.|
